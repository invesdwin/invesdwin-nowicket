This is a rewrite of the AJAX choice example from Wicket. This example demonstrates the @Eager annotation. Whenever the Manufacturer select box value is changed, it will submit the new value to server via AJAX and update the choices in the Model select box. Note that you can also put the @Eager annotation on the getter or setter, or even make a whole model @Eager by annotating a type. Also note the conventient handling of enums in the model, which do not need choice utility methods as long as you don't want to modify the available literals. Also we demonstrate status messages of different types and how exceptions can be thrown in buttons that get displayed as message boxes.
