This page demonstrates the famous hello world example for invesdwin-nowicket. It is a naked objects framework for wicket that enables developers to write less boilerplate wicket code during implementation of complex websites. Improving productivity by a measurable factor and making wicket more fun.

Use the menu at the top to look at other examples or have a look at the sourcecode on github. Something interesting there could be how the AExampleWebPage is used to build the menu and make it visible in all pages (by making all pages extend it).
